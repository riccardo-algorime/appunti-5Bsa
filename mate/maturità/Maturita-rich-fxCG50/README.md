# Maturita rich fx-CG50

Pacchetto di schede compatte per ripasso matematica, ottimizzate per Casio fx-CG50.

## Cartelle

- `jpg/`: file finali da copiare, JPEG baseline compatibili, 384 px di larghezza.
- `png/`: versioni PNG sorgenti renderizzate, piu nitide ma non necessarie.
- `_source/`: Markdown + LaTeX + renderer.

## Formato verificato

I JPG finali sono:

- JPEG baseline
- 3 componenti RGB
- non progressive
- larghezza 384 px

## Fonti di controllo argomenti

- Zanichelli, argomenti delle prove d'esame di matematica: geometria analitica nello spazio e calcolo della probabilita.
- Traccia MIM 2025, quesito piano-sfera.
- Traccia 2024 / raccolte svolte, quesito su proiezione ortogonale e intersezione retta-piano.
- Formulaio De Rosa e schede gia estratte localmente come base di confronto.
